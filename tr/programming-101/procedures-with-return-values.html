Dönüş değerlerine duyulan ihtiyaç

Şimdiye kadarki örnek prosedürlerimizin hepsinin yan etkileri oldu. Bu, ortamlarını bir şekilde etkiledikleri anlamına gelir, örneğin ekranda çıktıyı görüntüleme şekli println() .
var score  = (numCorrect/30) * 100;
println(score);
Çoğu zaman, ekranda hemen bir değer görüntülemek istemeyiz. Bunun yerine, bir hesaplamanın sonucunu hatırlamak ve gelecekte bir şekilde kullanmak istiyoruz. Bu durumda, yordama, onu çağıran koda bir değer döndürmesini söyleriz.
JavaScript dilinde, bir işlevden değerleri ve ardından döndürülecek bir ifade yazarak return döndürebiliriz.
function calcScore(numCorrect) {
   var score  = (numCorrect/30) * 100;
   return score;
}
Artık işlev bir değer döndürdüğüne göre, sonucunu aşağıdaki gibi bir değişkende depolamayı kullanabiliriz:
var sallyScore = calcScore(27);
var wilburScore = calcScore(24);
İşlevi doğrudan diğer işlevlerin içinde de çağırabiliriz:
println(calcScore(27));
println(calcScore(24));
Veya işlevi daha büyük bir ifadenin içinde çağırabiliriz:
println("Sally earned: " + calcScore(27));
println("Wilbur earned: " + calcScore(24));

Erken iadelere dikkat edin

Aşağıdaki kod bitinde bir hata vardır:
function calcLineSlope(x1, y1, x2, y2) {
    var yDiff = y2 - y1;
    var xDiff = x2 - x1;
    return slope;
    var slope = yDiff / xDiff;
}
Ne olduğunu görüyor musun? İşte bir ipucu: bilgisayar bir return deyim yürüttüğünde, işlevden hemen çıkar. İşlevden çıktıktan sonra, işlevde daha fazla kod yürütmez.
Bu, bilgisayarın yalnızca şu kodu yürüttüğü anlamına gelir:
function calcLineSlope(x1, y1, x2, y2) {
    var yDiff = y2 - y1;
    var xDiff = x2 - x1;
    return slope;
}
Eğimi hesaplayan çizgiye asla ulaşmaz, bu nedenle onu çağıran kişiye tanımsız bir değer döndürür. Ondan sonraki kod satırı "ölü kod", asla yürütülmeyecek koddur.
Kodun doğru sürümünde, return deyimi işlevdeki son satırdır:
function calcLineSlope(x1, y1, x2, y2) {
    var yDiff = y2 - y1;
    var xDiff = x2 - x1;    
    var slope = yDiff / xDiff;
    return slope;
}

Sözde kodda değerler döndür

[Sözde kod nedir?]
Bu sözde kod, adlandırılmış parameter1 iki parametre alan ve , programlama yönergelerini içeren ve parameter2ardından expression.
PROCEDURE name (parameter1, parameter2)
{
     <instructions>
    RETURN (expression)
}
İşte yordam ve çağrılar calcLineSlope için sözde kod:
PROCEDURE calcLineSlope (x1, y1, x2, y2) 
{
    yDiff ← y2 - y1
    xDiff ← x2 - x1    
    slope ← yDiff / xDiff
    RETURN slope
}

slope1 ← calcLineSlope(1, 2, 4, 5)
slope2 ← calcLineSlope(0, -1, 3, 10)
