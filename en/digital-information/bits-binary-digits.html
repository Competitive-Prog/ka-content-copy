Computers store information using bits. A bit (short for "binary digit") stores either the value 0\texttt{0}0start text, 0, end text or 1\texttt{1}1start text, 1, end text.

What fits in a bit?

A single bit can only represent two different values. That's not very much, but that's still enough to represent any two-valued state.

Sequences of bits
